# 课程介绍

- 首先呢，C++是大一的一个大类基础课（不过不同的院貌似会略微有区别），有3学分，在大一的课程中算一门很重要的课，学习难度和所需精力还是比较大的。

- 平时分由签到、章节作业、上机测试（比较水）组成；期末闭卷考试（有人肉编译题）

- TJU的C++课程难度不好评价，如果想拿到高分还是得努力理解和复习，但是老师也会捞人（也有人通了两个晚上70来分）

# 我的学习过程
